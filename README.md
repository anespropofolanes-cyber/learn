# # 國小學習地圖

一個給國小孩子（適齡 4–12 歲、核心 7 歲）的互動式學習網站，涵蓋 8 大科目、715 個學習主題、1275 道審核過的選擇題。**單一 HTML 檔、離線可開、不需伺服器**——所有圖片、程式、樣式都已內嵌在檔案裡。

## 功能

- **主頁**：太空封面＋四大區入口（各附簡介）。
- **主題與檢核**：8 科 × 領域收合，勾選「學會的表現」記錄進度（存在本機瀏覽器）。
- **學習地圖**：主題前置知識關係圖，點主題highlight它的先修脈絡。
- **練習**：今天 10 題、依科目／難度出題、可只出「已學會」主題；練習模式（答錯給提示、再試一次、看解析）與測驗模式；錯題本自動收錄。
- **家長**：今天題數、近 7 天練習量、各科答對率、最常錯的主題。

進度與練習紀錄存在瀏覽器 localStorage，不上傳、不需帳號；換裝置不會同步。

## 使用方式

直接用瀏覽器打開 `index.html` 即可，不需安裝或連網。

## 放上 GitHub Pages（可選）

1. 建立一個 GitHub repo，把 `index.html` 上傳到根目錄。
2. repo → Settings → Pages → Source 選 `main` 分支、根目錄，儲存。
3. 稍等一會，會得到一個 `https://<你的帳號>.github.io/<repo名>/` 的網址。

## 檔案說明

- `index.html`：**唯一必要的檔案**，完整可運作的網站（所有圖片、題目、程式皆已內嵌）。
- `source/`：**原始素材備份，網站執行不需要**，僅供日後擴充或重建用。
  - `source/images/`：封面與各分頁／各科橫幅的原始插圖（已壓縮內嵌進 HTML）。
  - `source/questions/`：AI（GPT／Codex）生成的題目 JSON 批次（batch1–22），已全部合併進 `index.html`。
  - `source/docs/`：出題規格書、擴充說明、待出題主題清單（TSV）、互動網頁規劃等文件。

### 如何擴充題目

參考 `source/docs/怎麼用GPT擴充題目.md` 與 `GPT出題規格書.md`：把規格書貼給 GPT／Claude，貼上想出題的主題（見 `待出題主題清單.tsv`），取得 JSON 後放進 `source/questions/`，再請協助者合併進 `index.html`。
